值得推荐的图文列表以及CSS布局类型切换

值得推荐的一种图文列表基本布局 经常碰到图文列表的排列，怎样既符合语义化标签又能扩展出多种表现形式呢。自己做过许多布局设计，总结出一点小经验和技巧。
- 列表排布不仅可以使用float排列，display:inline-block也是很好用的一个属性。

HTML代码如下： 
<div class="box">
  <ul>
    <li>
      <div class="item">
        <a href="url"><img src="src" alt=""/></a>

        <h3><a href="url">TITLE</a></h3>
        <p>Description</p>
      </div>
    </li>

    ... items ...
  </ul>
</div>

通过这个HTML layout，我们可以轻易拼出常用的图文列表表现形式：
* 第一种 常用的文章列表带缩略图布局；
* 第二种 图片带标题的固定长宽布局；
* 第三种 图片带标题定宽自动高度布局；
* 第四种 图片列表标题动画渐出（适当加一些效果很能讨好用户）
